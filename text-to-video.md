# 辩论稿 → 视频/音频对话：方案指南

## 方案总览

| 方案 | 难度 | 成本 | 效果 | 适合场景 |
|------|------|------|------|----------|
| NotebookLM（推荐） | ⭐ | 免费 | 自然对话 Podcast | 最快出活，辩论模式直出 |
| HeyGen | ⭐⭐ | $24/月起 | 数字人视频 | 社媒短视频、正式感 |
| Medeo | ⭐⭐ | 免费试用 | 多角色辩论视频 | TikTok/YouTube Shorts |

## 方案一：Google NotebookLM（最推荐 🔥）

**为什么首选**：NotebookLM 的 Audio Overview 有专门的 **Debate（辩论）模式**，两个 AI 主持人会自动围绕你的材料进行正反方辩论，效果极其自然。

### 操作步骤

1. 打开 https://notebooklm.google.com
2. 新建 Notebook，把辩论稿（gold-side.md、bitcoin-side.md、judge.md）全部上传为 Source
3. 点击右侧 **Studio** 面板 → **Audio Overview**
4. 点 **Customize（自定义）**：
   - 格式选 **Debate**
   - 语言选 **中文**
   - 在 prompt 里写：`请围绕"2026年应该投资黄金还是比特币"展开辩论，一方支持黄金，一方支持比特币，引用材料中的数据和论点`
5. 点 Generate，等几分钟即可

### 优势
- **免费**，效果惊艳
- 支持中文，语调自然、有停顿和互动感
- 自动提炼材料重点，不是机械念稿
- 可下载音频文件

### 局限
- 只有音频，没有画面（需要后期配视频画面）
- 两个主持人角色固定，无法自定义声音
- 偶尔会有轻微事实偏差

### 音频 → 视频的进阶操作
- 用 **剪映/CapCut** 导入音频，叠加股票图表、金价走势等画面素材
- 或用 **Canva** 做图文轮播视频配上该音频

## 方案二：HeyGen（数字人辩论视频）

**官网**：https://www.heygen.com/video/debate-video-maker

### 操作步骤

1. 注册 HeyGen，进入 Debate Video Maker
2. 将辩论稿转为对话脚本格式（角色A说…角色B说…）
3. 为【黄金】和【BTC】各选一个 AI 数字人形象
4. 分配不同的声音风格（如一个沉稳、一个犀利）
5. 添加品牌元素、背景
6. 生成视频，支持多种比例导出

### 优势
- 有逼真的数字人画面，唇形同步
- 支持中文语音
- 可品牌定制（Logo、配色）
- 直接出成品视频

### 局限
- 付费（$24/月起）
- 数字人表情仍有 AI 感
- 长辩论稿可能需要分段生成

## 方案三：Medeo（专用辩论视频生成器）

**官网**：https://www.medeo.app/debate-video-generator

### 操作步骤

1. 输入辩论主题和双方论点摘要
2. 选择 2-6 个 AI 角色，分配声音和性格
3. 点击 Generate
4. 用内置编辑器调整节奏、加字幕、配背景音乐
5. 导出竖版（TikTok）或横版（YouTube）

### 优势
- 专为辩论视频设计
- 支持多角色、表情手势
- 一站式出片

### 局限
- 中文支持待确认
- 效果依赖平台模型质量

---

## 方案四：Google Veo + Filmora（高质量制作）

### 操作步骤

1. 用 Claude/ChatGPT 将辩论稿改写为分镜脚本（场景描述 + 对话）
2. 打开 Filmora → Stock Media → AI Media → Text to Video
3. 选择 Google Veo 3.1 引擎
4. 粘贴分镜脚本，设置语言为中文
5. 生成后在时间线上精剪、拼接
6. 叠加数据图表、字幕、转场

### 优势
- 画面质量最高
- 唇形同步效果好
- 完整的视频编辑工具链

### 局限
- 流程较复杂，需要视频编辑基础
- Filmora 付费
- 需要自己组装分段视频

---

## 方案五：剪映 + AI 配音（国内方案）

### 操作步骤

1. 将辩论稿按角色拆分为逐句台词
2. 在剪映中使用 **AI 文字转语音**：
   - 【黄金】选一个沉稳男声
   - 【BTC】选一个年轻锐利男声
   - 【评委】选一个权威女声
3. 配合画面素材（金价走势图、比特币K线、新闻截图）
4. 加字幕、转场、背景音乐
5. 导出发布

### 优势
- 完全免费
- 国内网络直接使用
- 灵活度高，想怎么剪就怎么剪
- 适合抖音/B站/视频号

### 局限
- 没有数字人画面，纯声音+图文
- 手动工作量较大
- AI 声音缺少对话互动感

## 我的推荐路线

### 快速出活（30分钟内）
```
辩论稿 → NotebookLM (Debate 模式) → 音频文件 → 剪映配画面 → 成片
```

### 社媒短视频（1-2小时）
```
辩论稿 → 精简为3分钟对话脚本 → HeyGen/Medeo → 数字人辩论视频
```

### 高质量长视频（半天）
```
辩论稿 → 分镜脚本 → Veo/Filmora 生成片段 → 剪映精编 → 专业成片
```
